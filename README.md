# homebrew-easy-openclaw

[Easy OpenClaw](https://github.com/alittleshine/homebrew-easy-openclaw) 的 Homebrew Tap —— OpenClaw 全流程管理工具，面向中国大陆用户。

## 安装

```bash
brew tap alittleshine/easy-openclaw
brew install --cask easy-openclaw
```

## 更新

```bash
brew upgrade --cask easy-openclaw
```

## 卸载

```bash
brew uninstall --cask easy-openclaw
```
