# homebrew-easy-openclaw

[Easy OpenClaw](https://github.com/alittleshine/easy-openclaw-workspace) 的 Homebrew Tap。

## 安装（国内用户）

由于国内访问 GitHub 较慢，请按以下步骤操作：

**第 1 步：添加 Tap（使用代理 clone）**

```bash
git clone https://gh-proxy.com/https://github.com/alittleshine/homebrew-easy-openclaw /opt/homebrew/Library/Taps/alittleshine/homebrew-easy-openclaw
```

**第 2 步：安装**

```bash
brew install --cask easy-openclaw
```

> 如果代理不可用，可以替换为其他代理前缀：
> - `https://ghfast.top/https://github.com/...`
> - `https://ghproxy.cn/https://github.com/...`

## 更新

```bash
brew upgrade --cask easy-openclaw
```

## 卸载

```bash
brew uninstall --cask easy-openclaw
```
